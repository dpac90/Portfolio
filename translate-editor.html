Translate.com

Website Language Translation Editor

Period: May 2017 - October 2017
Company: Emerge Media
Role: Lead UI / UX Designer and Researcher
Project Type: Web Application

Introduction

  • Translate.com provides crowdsourced translations with help from a community of over 17,000 translators
  • Visual Editor is a web application that will allow translators as well as customers to translate text on websites while having the website in view

Problem Summary

  • When translation is performed on text that comes from websites, translators lack context as they only see the bare text
  • This led to poor translation quality for website translations
  • Customers had issues seeing the status of human translations on their websites that were ordered through Translate.com
